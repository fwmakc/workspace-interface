# Command Bar UI

Веб-интерфейс командной строки / лаунчер в стиле ОС. Чистый HTML/CSS/JS, без фреймворков.

## Быстрые факты

- **Язык:** JavaScript (ES5-совместимый стиль)
- **Архитектура:** Модульная, IIFE-паттерн под глобальным объектом `App`
- **Хранилище:** `localStorage` (префикс `cbui_`) — история, dock, тема Markdown, placeholder
- **Конфигурация:** Вынесена в `config/`, редактируется без изменения логики
- **Markdown:** Поддержка в чате с подсветкой синтаксиса и 3 темами (dark, light, dracula)
- **Chat:** Сообщения с выделением, копированием, навигацией `Alt+↑/↓/PgUp/PgDown/Home/End`
- **Режимы:** `auto`, `project`, `url`, `calc`, `notes`, `terminal` с авто-определением по тексту

## Структура проекта

```
index.html          → точка входа, загружает все CSS и JS
config/             → JSON-подобные конфиги (приветствия, фон, приложения, режимы)
css/                → 11 CSS-файлов по компонентам
js/                 → 15+ модулей (ядро, UI, ввод, статус, чат, терминал, markdown)
fonts/              → Noto Sans, Noto Sans Mono, Noto Color Emoji
img/                → Фоновые изображения
```

## Ключевые модули

| Модуль | Файл | Назначение |
|--------|------|-----------|
| Keyboard | `js/keyboard.js` | Хоткеи: `Enter`, `Ctrl+↑/↓`, `Shift+Space`, `Alt+1..6`, `Escape`, автофокус |
| Modes | `js/modes.js` | Переключение режимов, авто-определение по паттерну |
| Chat | `js/chat.js` | Сообщения, выделение, навигация, copy/delete |
| Markdown | `js/markdown.js` | Парсер Markdown + подсветка синтаксиса |
| Terminal | `js/terminal.js` | Виртуальная файловая система и команды |
| Commands | `js/commands.js` | Роутер калькулятора и терминала |
| Dock | `js/dock.js` | Закреплённые приложения, drag-and-drop |
| Apps Menu | `js/apps.js` | Меню приложений с поиском и навигацией |
| Attach | `js/attach.js` | Прикрепление файлов |
| Emoji | `js/emoji.js` | Пикер эмодзи |
